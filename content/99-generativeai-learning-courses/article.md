1. **Google Cloud Skills Boost - Introduction to Generative AI**
Good introduction into what GenerativeAI is and how it is described by providing examples. It's a short course and thus easy to consume to get the foundation. There is a quiz integrated with 5 questions that need to be answered in order to get the Introduction to Generative AI completion badge.
https://www.cloudskillsboost.google/course_templates/536

2. **Google Cloud Skills Boost - Introduction to Large Language Models**
This introductory course provides an overview of large language models (LLMs), including their applications, prompt tuning techniques, and how to utilize Google tools to build Gen AI applications.
https://www.cloudskillsboost.google/course_templates/539


3. **AWS - Foundations of Prompt Engineering**
This intermediate-level course provides a comprehensive overview of prompt engineering, covering the basics, advanced techniques, and best practices for designing effective prompts. It teaches you how to guard against prompt misuse and mitigate bias when interacting with large language models. The course is intended for prompt engineers, data scientists, and developers who want to enhance their skills in prompt design and application.https://explore.skillbuilder.aws/learn/course/external/view/elearning/17763/foundations-of-prompt-engineering


4. **ChatGPT Prompt Engineering for Developers**
This is a more in depths course for learning how to work with and writing prompts for ChatGPT.
https://www.deeplearning.ai/short-courses/chatgpt-prompt-engineering-for-developers/
